Funções Agregadas em SQL

  • AVG – A função AVG calcula a média dos valores de uma coluna numérica. Ela é útil para determinar a média de salários, idades ou qualquer outro tipo de dado numérico em uma tabela. 

    • //Exemplo:                                 
SELECT AVG(salario) 
FROM funcionarios;

  • COUNT – A função COUNT conta o número de registros que atendem a uma condição específica. Quando usada com COUNT(*), ela conta todas as linhas da tabela. 

    • //Exemplo:   
SELECT COUNT(*) 
FROM alunos 
WHERE idade > 18;

  • SUM – A função SUM soma todos os valores de uma coluna numérica. Pode ser usada, por exemplo, para calcular o total de vendas de uma empresa ou a soma de pontos de um time. 

    • //Exemplo:   
SELECT SUM(vendas) 
FROM produtos;

  • MAX – A função MAX retorna o maior valor encontrado em uma coluna numérica. Ela é útil para identificar o maior valor em uma lista de dados, como o maior salário ou o maior preço de produto. 

    • //Exemplo:   
SELECT MAX(preco) 
FROM produtos;

  • MIN – A função MIN retorna o menor valor encontrado em uma coluna. Pode ser utilizada para encontrar o valor mais baixo de uma lista, como o preço mais baixo ou a menor idade em uma tabela. 

    • //Exemplo:   
SELECT MIN(preco) 
FROM produtos;

As funções agregadas são muito úteis para resumir grandes volumes de dados. Elas permitem realizar cálculos como média, soma, contagem, máximo e mínimo de maneira eficiente, sem precisar processar manualmente grandes quantidades de registros.

Essas funções são frequentemente usadas junto com as cláusulas GROUP BY para agrupar os dados e HAVING para filtrar resultados agregados. Assim, você pode obter insights importantes de seus dados de forma rápida e prática.
