Comandos DML

  • SELECT – O comando SELECT é utilizado para recuperar dados de uma tabela em um banco de dados. Ele permite visualizar as informações armazenadas, podendo filtrar, agrupar ou ordenar os dados de diversas formas. 

    • //Exemplo:                                
SELECT nome, idade 
FROM alunos 
WHERE idade > 18;

  • INSERT – O comando INSERT é usado para adicionar novos registros a uma tabela. Você deve fornecer os valores correspondentes às colunas definidas. 

    • //Exemplo:   
INSERT INTO alunos (nome, idade) 
VALUES ('Ronaldo', 22);

  • UPDATE – O comando UPDATE permite modificar os dados existentes em uma tabela. É útil, por exemplo, para corrigir informações ou atualizar registros. 

    • //Exemplo:   
UPDATE alunos 
SET idade = 19 
WHERE nome = 'Pedro';

  • DELETE – O comando DELETE serve para remover registros de uma tabela. Ele deve sempre ser usado com cuidado, geralmente acompanhado de uma condição. 

    • //Exemplo:   
DELETE FROM alunos 
WHERE nome = 'Leo';

Os comandos DML (Data Manipulation Language) são fundamentais para trabalhar com bancos de dados relacionais. O SELECT é utilizado para consultar e visualizar informações; o INSERT insere novos registros; o UPDATE atualiza dados já existentes; e o DELETE remove informações de forma definitiva. Dominar esses comandos é essencial para quem deseja manipular dados de maneira eficiente, garantindo consultas rápidas, inserções corretas e manutenção adequada das tabelas em um banco de dados.
