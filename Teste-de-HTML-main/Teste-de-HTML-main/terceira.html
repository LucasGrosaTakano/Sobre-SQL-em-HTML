Introdução

Comandos DCL

  • GRANT – Dar privilégios a um usuário ou role para acessar objetos do banco (tabelas, views, procedimentos, etc.) ou administrar o banco como um todo.

  • REVOKE – Retirar permissões concedidas com GRANT.

  • DENY – O comando DENY bloqueia explicitamente um privilégio, mesmo que venha de uma herança (por exemplo, via roles).

Tipos de Dados no MySQL

No MySQL, os dados são divididos em três categorias: numéricos, strings (texto e binário) e data/hora. Os numéricos incluem inteiros e decimais, enquanto as strings incluem textos fixos e variáveis, além de BLOBs. Os tipos de data/hora incluem DATE, DATETIME e TIMESTAMP.
